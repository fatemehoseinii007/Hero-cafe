## ویژگی‌ها

- طراحی واکنش‌گرا (Responsive)
- نوار ناوبری (Navigation Bar) مدرن
- بخش قهرمان (Hero Section) جذاب
- بخش درباره ما (About Us)
- طراحی زیبا و مدرن

## تکنولوژی‌ها

- HTML5
- CSS3 (Flexbox, Grid, Animations)

## نحوه مشاهده پروژه
